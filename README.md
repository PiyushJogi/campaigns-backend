**Usage Instructions:**

##.env file : Create .env file at root of the backend directory
```bash
PORT=4000
DATABASE_URL=mongodb://localhost:27017
```
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
